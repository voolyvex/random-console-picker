## Installation
### Windows
- Download `Random.Console.Picker.Setup.exe`
- Run the installer
- Launch from desktop shortcut or start menu

### macOS
- Download `Random.Console.Picker.dmg`
- Mount the disk image
- Drag to Applications folder
- Launch from Applications
